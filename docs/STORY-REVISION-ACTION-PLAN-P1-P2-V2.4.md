# CariDream P1/P2 Story Revision Plan

## Canon Edition v2.4

**Scope:** 42 stories: 25 P1 and 17 P2  
**Excluded:** The Hidden Waltz of Jarabacoa (P3)  
**Preservation policy:** Correct metadata and series placement before changing narrated prose. Existing audio should remain unless the spoken text itself must change.

## Canon Rule Key

| Code | Canon rule affected |
| --- | --- |
| **NT** | Never Teach the Lesson: meaning should emerge through experience rather than direct instruction. |
| **CF** | Caribbean First: the story needs a specific, respectful Caribbean place and cultural perspective. |
| **IR** | Island Rule: each island must contribute a distinct identity without duplicating or overshadowing another. |
| **CH** | Canon Hierarchy: Guardians, Great Spirits, Folk Figures, Island Legends, and landscapes must retain separate roles. |
| **MS** | Major Series and continuity governance: stories must be assigned to a coherent series and recurring role. |
| **LV** | Lore Vault Rule: living folklore, spiritual traditions, and culturally sensitive interpretations require research before canon promotion. |

## Audio Decision Key

| Decision | Meaning |
| --- | --- |
| **No** | Metadata, description, ordering, or series assignment can fix the issue. Keep the current audio. |
| **Not required now** | A prose enhancement would help, but it can wait for a later remaster. Keep the current audio for beta. |
| **Yes, after approval** | The spoken lesson, role, or cultural framing must change. Approve revised text first, then regenerate once. |
| **Revise before first generation** | Apply the text fix before generating pending narration so no MP3 is wasted. |

## Recommended Fix Order

1. **Metadata-only corrections first:** Haiti's Jacmel series, Bahamas Sea Turtle series, Dominican subseries, Saint Lucia rainforest split, Jamaica ordering, and canon-role notes. These preserve all existing audio.
2. **Cultural review second:** Haiti's historical stories and Trinidad's Silk Cotton story. Do not rewrite or regenerate until the factual and cultural framing is approved.
3. **Short spoken-text edits third:** Remove only explicit moral lines from Anansi, cactus, rain, turtle, and Papa Bois stories. Approve each final text before generating replacement audio.
4. **Pending Papa Bois episodes before narration:** Fix Moonlit River and Firefly Path before their first MP3 generation.
5. **Defer optional remasters:** Keep P2 audio and the overlapping Blue Mountain recordings during beta. Use listener feedback to decide whether deeper rewrites are worthwhile.

## Preservation Summary

- **42 stories listed**
- **25 stories can be corrected without regenerating audio**
- **1 story can keep its audio until an optional later remaster**
- **14 stories require regeneration only after revised text is approved**
- **2 pending Papa Bois stories should be revised before their first audio generation**

This approach resolves series and canon-identity problems immediately while limiting narration work to stories whose spoken content genuinely conflicts with Canon Edition v2.4.
